Created by Conor and Ray, 2FAN (Two Factor Authentication Notifier) strives to help all netizens be more secure on the web.

Thanks to the folks at twofactorauth.org for opensourcing their project, which allowed us to build this tool.

Also, thanks to Blair Adams, Perametade Games, anbileru adaleru for allowing us to use their icons through thenounproject.com as well as the Google Design team for open sourcing their icons.

If you are interested in jumping into the code for 2FAN, we've open sourced everything on our Github repo here.
